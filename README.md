# Simple Blog Management System

## Overview
This project is a Full-Stack Blog Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, and JSON Server. The system allows users to register, login, create posts, edit posts, delete posts, search posts, and categorize blog entries.

## Features
- User Registration
- User Login Authentication
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- Search Posts
- Category Filtering
- Posted Date
- Responsive Dark UI Design

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- JSON Server

## Installation

1. Clone the repository
```bash
git clone YOUR_GITHUB_LINK
```

2. Install dependencies
```bash
npm install
```

3. Run the server
```bash
node server.js
```

4. Open frontend
Run index.html using Live Server.

## API Base URL
```text
http://localhost:3000/api
```
